BerryTweaks

Installation

Click here to install if you have a userscript addon, such as Tampermonkey for Chrome or Greasemonkey for Firefox.

Alternatively, load the script temporarily by running this in your JavaScript console: 

$.getScript('https://atte.fi/berrytweaks/js/init.js');

Usage

No tweaks are enabled by default. You can find new checkboxes in BerryTube's settings for enabling the tweaks you want. Your choices are saved per-browser.

The code for a tweak is only loaded if it's enabled, so the fewer you enable the lighter the script will be.
